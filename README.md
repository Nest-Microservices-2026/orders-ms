## Project setup

```bash
$ npm install
$ docker compose up
$ npm install prisma --save-dev
$ npx prisma
$ npx prisma -v
$ npx prisma init
$ npx prisma migrate dev --name init
$ npm install @prisma/client
$ npx prisma generate
$ npm install @prisma/adapter-pg

```

## Compile and run the project

```bash
# watch mode
$ npm run start:dev

```
